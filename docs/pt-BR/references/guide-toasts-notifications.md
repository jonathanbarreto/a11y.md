# Guia de Toasts & Notificações

> **Escopo:** Feedback Dinâmico

## Regras Centrais
1. **Alertas:** Erros críticos MUST usar `role="alert"` (assertive).
2. **Status:** Mensagens comuns MUST usar `role="status"` (polite).
3. **Auto-fechamento:** Evite fechar automaticamente toasts com ações ou muito texto. Dê controle ao usuário.
