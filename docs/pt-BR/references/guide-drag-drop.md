# Guia de Acessibilidade em Drag and Drop

> **Escopo:** Listas Interativas e Boards

## Regras Centrais
1. **Alternativa de Teclado:** Toda ação de arrastar/soltar MUST ter equivalente via teclado (ex: Espaço para selecionar, setas para mover).
2. **Anúncios Aria-live:** Leitores de tela MUST ser informados das ações usando `aria-live`.
3. **Dicas Visuais:** O foco visível MUST acompanhar o item.
